## üöÄ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn package manager
- Modern web browser with WebGL support

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/your-org/vers3dynamics.git
cd vers3dynamics
```

2. **Install dependencies**
```bash
npm install
```

3. **Start the development server**
```bash
npm run dev
```

4. **Open your browser**
```
http://localhost:5173
```

### Production Build

```bash
npm run build
npm run preview
```
